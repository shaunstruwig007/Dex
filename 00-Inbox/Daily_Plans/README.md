# Daily Plans

Directory kept for **path contracts** and older hooks that expect `00-Inbox/Daily_Plans/`.

**Canonical daily plans** from `/daily-plan` are written to **`07-Archives/Plans/YYYY-MM-DD.md`** (see the daily-plan skill). Use that path for new plans and reviews.

If you generate a plan here instead, use one file per day with priorities, meetings, and execution notes.

This folder must exist for compatibility; it may be empty when you only use `07-Archives/Plans/`.
