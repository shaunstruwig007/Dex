# Weekly synthesis — exec & product pack

**Purpose:** One markdown file per week for **Product** and **Executive** — distilled from daily briefs + ingest + `Market_and_deal_signals.md`.

**How to produce:** Invoke **`/weekly-exec-intel`** (after running **`bash .scripts/market-intelligence/run-weekly-intel-fetch.sh`** for automated RSS, or your usual ingest).

**Template:** [_template_weekly_exec_brief.md](./_template_weekly_exec_brief.md) → save as `YYYY-MM-DD_weekly_exec_brief.md` (week-ending date in the filename or title).

**Difference from daily:** Fewer items, stronger “so what,” explicit product asks and leadership narrative — suitable for a weekly forum or email.
