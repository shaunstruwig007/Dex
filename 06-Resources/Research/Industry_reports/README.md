## How to use this folder

1. **As a Felix input** — `/felix-custom` Step 3 reads `<sector>/*.pdf` filenames + the README; deep-dives on `--deep`.
2. **As an ICP / Industry Truths input** — when authoring `System/icp.md` or running `/industry-truths`, pull the most recent sector report for evidence.
3. **As S3B per-card context** — `/moneypenny-custom` *(formerly `/pdlc-discovery-research-custom`; S3B — 2026-04-24 persona rename)* can scope to the sector(s) implied by the initiative's `targetUsers` brief field.
4. **As account-level intel** — when prepping for a CSM call with a Mining or Retail customer, the sector folder gives sector-wide framing.
