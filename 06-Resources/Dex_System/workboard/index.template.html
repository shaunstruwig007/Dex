PDLC doc process

PRDs and AC move Discovery → Design → Develop → Deploy. Drag cards between columns — order is saved in browser localStorage (dex-pdlc-doc-v1). Edit seed data in pdlc-doc-items.json + run python3 build_index.py. Idea for later: orchestration hooks on stage change (Slack, branch, checklist).

Tasks server: python3 workboard_server.py — task swim lanes update work-items.json + 03-Tasks/Tasks.md.   Roadmap PRD panel: right sidebar + visual edit (HTML→MD via Turndown) / raw tab; GET/POST /api/prd, rollback, POST /api/prd/create, GET /api/prd/future-created; backups prd-backups/.   Roadmap tile summary + PDLC override: localStorage dex-roadmap-card-meta-v1.   PDLC doc board: localStorage only (no server API yet).   Roadmap Then ↔ Future order: dex-roadmap-then-future-v3.   Offline tasks only: python3 -m http.server 8765 (PRD save unavailable).
